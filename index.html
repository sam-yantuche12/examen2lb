Guia De Trabajo De Comandos Git

git init:

inicia nuevos repositorio git local en el directorio actual

git add:

agregamos todos los archivos al area de preparacion (staging)

git push:

Envia los commits locales al repositorio remoto

git status:

Muestra el estado actual del repositorio, archivo modificados y no rastreados

git commit -m "mensaje":

Guarda los cambios agegados con un mensaje descriptivo

git pull:

obtiene y fusiona los cambios del repositorio remoto
